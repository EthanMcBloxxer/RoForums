--- --- {% capture content %} # RoForums RoForums is a Roblox "game" created in order to try to make or recreate the Roblox Forums. As of writing, 3/3/2021, the project isn't anywhere near being finished. We aren't using Rojo yet, but we do have an rbxlx file you can download now and start editing. If you know anything about Roblox Studio, I recommend cloning the repo, taking a look at our project boards on GitHub and doing a task there, then opening `roforums.rbxlx`. You just have to edit it, then make a pull request explaining what you did. There isn't much other information at this point, because this page will be replaced with documentation and/or api documentation. I'm hoping to turn this into something you can create your own of or embed in an existing game like you can with [Mastodon](https://joinmastodon.org/). A quick peek at the UI is available at [`studio.png`](https://raw.github.com/EthanMcBloxxer/RoForums/brand/studio.png) on the [`brand`](https://github.com/EthanMcBloxxer/RoForums/tree/brand) branch of the GitHub repository.
